# Admin Ropa - Panel de gestión para emprendimientos

Esta es una página administrativa lista para publicar en GitHub Pages. Permite administrar:

- Inversión
- Ventas
- Gastos
- Nómina

## Cómo usar

1. Abre `index.html` en el navegador.
2. Navega entre los secciones del panel.
3. Usa los botones "Agregar registro" para insertar filas.
4. Los datos se guardan automáticamente en el navegador con `localStorage`.
5. Puedes exportar todos los datos en formato CSV con el botón "Exportar todo CSV".

## Publicación en GitHub Pages

1. Crea un repositorio en GitHub.
2. Sube estos archivos.
3. Activa GitHub Pages desde la rama principal.
4. Usa la URL proporcionada por GitHub.

> Nota: el almacenamiento es local al navegador. Si deseas mantener un respaldo externo, descarga el CSV regularmente.
